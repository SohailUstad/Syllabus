Operators & Keywords
Operator in Java is a symbol that is used to perform operations. For example: +, -, *, / etc. There are many types of operators in Java which are given below:
  • Arithmetic Operator
  • Comparison Operator
  • Logical Operator
  • Assignment Operator
1) Arithmetic Operator 
                +       Additive operator (also used
                        for String concatenation)
                -       Subtraction operator
                *       Multiplication operator
                /       Division operator
                %       Remainder operator
2) Comparison Operator 
                ==      Equal to
                !=      Not equal to
                >       Greater than
                >=      Greater than or equal to
                <       Less than
                <=      Less than or equal to
3) Logical Operator 
                &&      AND
                ||      OR
                !       NOT
4) Assignment Operator 
                =        Equalto operator
                +=       Plus Equalto operator
                -=       Minus Equalto operator
                *=       Multiply Equalto operator
                /=       Divide Equalto operator
                %=       Remainder Equalto operator
